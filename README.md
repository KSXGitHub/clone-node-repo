# clone-node-repo
My own script to clone my own repo at home

## Usage

### Command line

#### Windows Command Prompt

```batch
set CLONED_NODE_REPO=<repo-name>
clone-node-repo
```

#### Bash shell

```bash
CLONED_NODE_REPO=<repo-name>
clone-node-repo
```

### JavaScript

```javascript
require('clone-node-repo')(REPO_NAME);
```
